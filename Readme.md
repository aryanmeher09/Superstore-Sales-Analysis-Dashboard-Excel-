# Project Overview

This project analyzes the **Superstore Sales dataset** using **Microsoft Excel** to uncover key business insights related to sales performance, profitability, and customer segments.

An interactive dashboard was created to help stakeholders monitor **Key Performance Indicators (KPIs), product performance, and sales trends**.

This project demonstrates how Excel can be used for **data analysis, visualization, and business intelligence reporting**.

---

# Business Problem

Retail companies generate large amounts of sales data but often struggle to quickly identify:

* Which **product categories generate the most profit**
* Which **products drive the highest sales**
* How **sales change over time**
* Which **customer segments contribute the most revenue**

This dashboard transforms raw data into **clear and actionable business insights**.

# Dataset Description

The dataset contains transactional sales records including:

* Order Date
* Region
* Category
* Sub-Category
* Sales
* Profit
* Customer Segment
* Discount

This information allows analysis of **sales performance across different product categories and customer segments**.

# Business Questions Answered

This dashboard helps answer the following business questions:

1. Which **product categories generate the highest profit?**
2. Which **sub-categories drive the most sales revenue?**
3. How does **sales performance change across months?**
4. Which **customer segment contributes the most sales?**
5. What is the **overall profit margin of the business?**

---

# Key Business Insights

* The company generated **$2.29M in total sales revenue**.
* Overall **profit margin is 12%**.
* **Technology category contributes the highest profit**.
* Sub-categories like **Phones and Chairs generate the highest sales**.
* The **Consumer segment contributes the largest share of sales**.
* Sales show **strong performance towards the end of the year**.

# Methodology

* Designed an interactive dashboard
* Added slicers for filtering by **Region and Year**
